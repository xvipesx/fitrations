## FitRations
FitRations is a self hosted nutrition manager application that puts you in control of your food and goals. As it is web-based, it acts as a website for easy management of your nutritional journal and can be hosted locally or via a VPS. 

## Tech Stack
The backend for FitRations is written in Python using FastAPI. Data related to foods, your personal journal and goals are conveniently in a single SQLite database file that you can backup as required. The frontend is written using React.
